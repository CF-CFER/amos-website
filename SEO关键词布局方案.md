## 五、关键词密度控制

### 最佳密度范围

| 元素 | 关键词密度 | 说明 |
|------|-----------|------|
| Title | 核心词出现1-2次 | 标题长度60字符内 |
| Meta Description | 核心词出现1-2次 | 前80字最重要 |
| H1 | 核心词出现1次 | 只能有1个H1 |
| H2-H3 | 长尾词出现 | 每个H2带1个关键词 |
| 正文 | 1.5%-2.5% | 自然出现，不堆砌 |
| Alt标签 | 关键词出现 | 图片说明带关键词 |
| 锚文本 | 长尾词 | 内部链接用关键词 |

### 密度检查工具

```bash
# 手动检查公式
关键词密度 = (关键词出现次数 / 总字数) × 100%

# 示例
文章总字数：1000字
"碳纤维"出现次数：15次
密度 = 15/1000 × 100% = 1.5% ✅ 正常

如果"碳纤维"出现30次
密度 = 30/1000 × 100% = 3.0% ⚠️ 偏高，可能被视为堆砌
```

在正文中自然融入LSI词，告诉搜索引擎：**这是真正懂行的内容**。

```

"碳烯技术生产的碳纤维丝（Carbon Fiber Filament）采用优质PAN基前驱体，
经高温碳化制成。我们的碳纤维束（Tow）规格齐全，从1K到24K均可提供。
这些碳丝（Carbon Yarn）表面经过专门上浆处理，确保与环氧树脂、乙烯基酯
树脂的良好结合。"

这段文字中融入了：碳纤维丝、PAN基、碳纤维束、碳丝、上浆处理、环氧树脂
→ 搜索引擎判断：这是专业的碳纤维材料内容 ✅
```
