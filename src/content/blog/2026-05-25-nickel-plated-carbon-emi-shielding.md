## 镀镍碳纤维简介

镀镍碳纤维是在碳纤维表面镀上一层镍金属，赋予碳纤维优异的导电性和电磁屏蔽性能。

## 三种形态全覆盖

碳烯技术独家提供镀镍碳纤维的三种形态：

### 镀镍碳纤维丝
- 规格：1K / 3K / 6K / 12K / 24K
- 镍层厚度：0.2-2.0 μm
- 表面电阻率：< 0.1 Ω/sq
- 屏蔽效能：> 60 dB

### 镀镍碳纤维粉
- 目数：50目 / 100目 / 200目 / 300目
- 镍层厚度：0.5-2.0 μm
- 体积电阻率：10⁻²-10¹ Ω·cm
- 屏蔽效能：40-60 dB

### 镀镍短切碳纤维
- 长度：3mm / 6mm / 10mm / 20mm / 50mm
- 应用：导电塑料改性
- 特点：注塑分散性好
